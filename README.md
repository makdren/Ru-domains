# Это не офицальный репозиторий v2fly

Этот проект управляет списком доменов, которые будут использоваться в качестве геосайтов для маршрутизации.

## Ссылка для скачивания:

- **dlc.dat**：[https://github.com/makdren/Ru-domains/releases/download/new/dlc.dat](https://github.com/makdren/Ru-domains/releases/download/new/dlc.dat)

## Дополнительная информация
В данный список будут добовляться российсуие компании, если вы хотите добавть какую-то компанию, напишите в issues
Также из списка удалены некоторые непопулярные в России домены, для уменьшения размера файла.
